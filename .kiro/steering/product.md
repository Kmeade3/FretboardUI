# Product Overview

Guitar Fretboard UI - An interactive fretboard visualization tool for guitarists and musicians.

## Purpose

Helps users visualize and explore musical scales across different guitar tunings and string configurations. Users can select from various tunings (6, 7, 8-string guitars and ukulele), choose scales (major, minor, pentatonic, blues, modes), and see which notes belong to the selected scale on the fretboard.

## Key Features

- Support for multiple instrument types (6/7/8-string guitars, ukulele)
- Extensive tuning library (standard, drop, open tunings)
- Scale visualization with 13+ scale types including modes
- Interactive fretboard with clickable notes
- Toggle scale highlighting on/off
- Configurable fret range (18-24 frets)
